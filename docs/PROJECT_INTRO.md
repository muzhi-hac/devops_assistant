# Parallax OpsPilot - 项目介绍

## 🎯 项目概述

**Parallax OpsPilot** 是一个基于 AI 的 DevOps 命令行助手，通过自然语言生成精确的 shell 命令。项目使用分布式推理技术，支持在多个设备间共享 GPU 资源，实现高效的本地 AI 推理，保护用户隐私。

## 💡 创新点

### 1. 分布式推理架构
- 支持跨设备（Mac、Linux、Azure VM）分布式运行模型
- 智能模型分片，充分利用多设备 GPU 资源
- 降低单设备计算压力，提升推理速度

### 2. 隐私保护
- 所有推理在本地完成，数据不上传云端
- 支持私有部署，完全控制数据流向
- 符合企业级安全要求

### 3. 智能命令生成
- 自动检测操作系统和 Shell 类型
- 生成兼容的命令，避免跨平台问题
- 危险命令自动警告，提升安全性

### 4. 优秀的用户体验
- 实时流式输出，即时反馈
- 美观的终端界面（Rich 库）
- 交互式配置，简单易用

## 🏆 技术亮点

### 核心技术栈
- **分布式推理**: Parallax 框架实现跨设备模型分片
- **流式处理**: 实时显示生成过程，提升用户体验

## 🔧 技术实现

### 1. 命令生成流程

```
用户输入 → 系统信息检测 → Parallax API → 流式输出 → 命令提取 → 用户交互
```

### 2. 分布式架构

```
Mac (Scheduler) ←→ Azure VM (Worker)
     ↓                    ↓
  模型层 1-10         模型层 11-20
     ↓                    ↓
     └────────→ 完整模型推理 ←────────┘
```

### 3. 安全机制

- 命令验证和清理
- 危险命令检测
- 人工确认机制
- 本地数据处理
